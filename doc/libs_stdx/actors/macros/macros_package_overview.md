# stdx.actors.macros

## 功能介绍

该包为用户提供了一个 [@Actor](./macros_package_api/macros_package_macros.md#actor-宏) 宏，用于声明一个 actor；以及一个 [@Receiver](./macros_package_api/macros_package_macros.md#receiver-宏) 宏，用于声明一个接收函数。

## API 列表

### 宏

|              宏          |           功能           |
| --------------------------- | ------------------------ |
| [Actor](./macros_package_api/macros_package_macros.md#actor-宏)     | 声明一个 actor。 |
| [Receiver](./macros_package_api/macros_package_macros.md#receiver-宏)     | 声明一个接收函数。 |
