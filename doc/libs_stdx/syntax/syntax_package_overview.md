# stdx.syntax

## 功能介绍

syntax 包主要包含了仓颉源码的语法解析器和仓颉语法树节点，提供语法解析函数。可以将指定路径的仓颉文件或包解析为抽象语法树（Abstract Syntax Tree）节点对象。

> **注意：**
>
> syntax 包需要依赖 `cangjie/tools/lib` 目录下的 `libcangjie-lsp` 动态库。
